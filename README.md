# Amanda Castro - Designer Gráfica | Portfólio 🎨

Este é o portfólio digital imersivo e interativo construído para Amanda Castro, Designer Gráfica com 5+ anos de atuação no Brasil. O projeto serve como um ecossistema para destacar sua identidade áudio-visual, e exibe de modo elegante seus trabalhos primordiais na área de Design, Social Media, Branding e Edição em Layouts Editorias.

## 🚀 Tecnologias Adotadas

A base deste projeto foi consolidada utilizando ferramentas avançadas na vanguarda do Web Design moderno:
- **React 18 & R3F (React Three Fiber)**: Para a estrutura Single Page e a galeria inovadora que se baseia inteiramente em 3D.
- **Framer Motion**: Usado para aplicar os cálculos e animações polidas durante as aberturas das janelas de informações interativas (*Sobre*, *Mídia*).
- **Tailwind CSS**: Garantiu padronização instantânea na adoção do padrão visual "Glassmorphism" — janelas transparentes sobre fundos elegantes na paleta Zinc.

## 📁 Seções Importantes
1. **Rolagem Parallax (Galeria Z)**: Ao descer a página, os projetos chegam pela tela em efeito tridimensional com luzes dinâmicas.
2. **Layout Living System**: Existe uma documentação viva anexada (`design-system.html`) documentando rigorosamente todas as escolhas de fonte, transições e uso da biblioteca de estilo de cores.
3. **Overlays Minimalistas**: Para que a navegação nunca perca a imersão visual, todas as interações ("Sobre a Amanda", "Detalhes da Obra", "Serviços Oferecidos") ocorrem por cima da tela do canvas.
